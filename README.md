# Expensely - Expense Management System

A modern, full-stack expense management application built with Node.js, Express, MongoDB, and Vanilla HTML/CSS/JS.



## Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB (Mongoose)

## Project Structure
- `backend/`: Express server, database config, and API routes.
- `frontend/`: Client-side UI and logic.

## Getting Started

### Prerequisites
- Node.js installed
- MongoDB installed and running

### Backend Setup
1. `cd backend`
2. `npm install`
3. Create a `.env` file (template provided)
4. `npm run dev`

### Frontend Setup
1. Open `frontend/index.html` in your browser.
